font-palette

A propriedade CSS permite especificar uma das muitas paletas contidas em uma fonte que um agente do usuário deve usar para a fonte. Os usuários também podem substituir os valores em uma paleta ou criar uma nova paleta usando a regra @font-palette-values .font-palette

Sintaxe

valores

Especifica a paleta de cores padrão ou a colorização do glifo padrão (definida pelo fabricante da fonte) a ser usada para a fonte. Com essa configuração, a paleta na fonte no índice 0 é renderizada.

Especifica a primeira paleta na fonte que corresponde a 'light' a ser usada para a fonte. Algumas fontes contêm metadados que identificam uma paleta como aplicável a um fundo claro (próximo ao branco). Se uma fonte não tiver esses metadados, o lightvalor se comportará como normal.

Especifica a primeira paleta na fonte que corresponde a 'dark' a ser usada para a fonte. Algumas fontes contêm metadados que identificam uma paleta como aplicável a um fundo escuro (próximo ao preto). Se uma fonte não tiver esses metadados, o valor se comportará como normal.
