Usando consultas de contêiner

Para usar consultas de contêiner, você precisa declarar um contexto de contenção em um elemento para que o navegador saiba que você pode querer consultar as dimensões desse contêiner posteriormente. Para fazer isso, use a container-typepropriedade com um valor de size, inline-sizeou normal.

A consulta será baseada nas dimensões inline e de bloco do contêiner. Aplica contenção de layout, estilo e tamanho ao contêiner.

O elemento não é um contêiner de consulta para consultas de tamanho de contêiner, mas permanece um contêiner de consulta para consultas de estilo de contêiner. Considere o seguinte exemplo de um componente de cartão para uma postagem de blog com um título e algum texto:

In the previous section, a container query applied styles based on the nearest ancestor with a containment context. It's possible to give a containment context a name using the container-name property. Once named, the name can be used in a @container query so as to target a specific container.
