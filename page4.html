API de Desempenho

A Performance API é um grupo de padrões usados ​​para medir o desempenho de aplicações web.

Conceitos e uso

Para garantir que os aplicativos da Web sejam rápidos, é importante medir e analisar várias métricas de desempenho. A API de desempenho fornece métricas integradas importantes e a capacidade de adicionar suas próprias medições à linha do tempo de desempenho do navegador. A linha do tempo de desempenho contém carimbos de data/hora de alta precisão e pode ser exibida nas ferramentas do desenvolvedor. Você também pode enviar seus dados para pontos finais de análise para registrar métricas de desempenho ao longo do tempo.
